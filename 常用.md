# dos命令查看杀死Windows进程

```bash
#查看所有端口占用的情况
netstat -ano
#查看特定端口占用的情况
netstat -ano|findstr "8081"
#通过进程id查看对应的进程名称
tasklist|findstr "16066"
#杀死进程命令
taskkill /f /t /im "16066"("java.")
```
